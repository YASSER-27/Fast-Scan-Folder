# install
Manual Installation (Windows)
To use the scan command from any terminal (CMD, PowerShell), follow these steps:

Prepare the Directory: Move the scan.exe file to a permanent folder (e.g., C:\my_tools).

System Properties: Open the Start Menu, search for "Edit the system environment variables", and select it.

Environment Variables: Click the Environment Variables button near the bottom right.

Edit Path: Under the System variables section, locate the Path variable, select it, and click Edit.

Add New Entry: Click New and paste the full path of the folder where you placed the file (e.g., C:\my_tools).

Save Changes: Click OK on all open windows to apply the changes.
